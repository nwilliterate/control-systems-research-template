# docs/literature/ — Literature

A place to collect prior-work summaries and paper reading notes.

## Rules
- One note per paper: copy `reading_notes_template.md` → `note_authorYEAR.md`.
- Register every citation as a BibTeX entry in `../../references/references.bib`,
  and put its citation key (e.g. `author2024keyword`) in the note for traceability.
- For a topic-level synthesis, write a separate `survey_<topic>.md`.

## ARS integration (optional)
If the academic-research-skills plugin is available:
- `/ars-lit-review` — annotated bibliography in paper format
- `/ars-citation-check` — citation error report
